# Web端图像标注工具 - 产品需求文档 (PRD)

## 1. 项目概述

### 1.1 项目名称
Web端图像标注工具

### 1.2 项目目标
开发一个基于 Streamlit 的 Web 端图像标注工具，提供边缘检测、人脸检测、目标检测以及丰富的图像处理功能，支持实时参数调节和结果导出。

### 1.3 目标用户
- 计算机视觉开发者和研究者
- 需要快速进行图像标注的用户
- 学习图像处理技术的学生

### 1.4 使用场景
- 快速预览图像处理算法效果
- 教学演示图像处理技术
- 图像数据标注辅助工具
- 图像增强与处理

---

## 2. 功能需求

### 2.1 检测模块

| 需求编号 | 功能名称 | 功能描述 | 优先级 |
| :--- | :--- | :--- | :--- |
| FR-001 | 边缘检测 (Canny) | 使用 OpenCV 的 Canny 算法进行边缘检测 | 高 |
| FR-002 | 人脸检测 (Haar Cascades) | 使用 Haar 级联分类器检测图像中的人脸，支持多分类器融合、眼睛检测、肤色过滤 | 高 |
| FR-003 | 目标检测 (DNN) | 使用 OpenCV DNN 模块加载 MobileNet-SSD 预训练模型进行目标检测 | 高 |

### 2.2 基础图像处理模块

| 需求编号 | 功能名称 | 功能描述 | 优先级 |
| :--- | :--- | :--- | :--- |
| FR-004 | 色彩空间转换 | 支持灰度、HSV、RGB 色彩空间转换 | 中 |
| FR-005 | 几何变换 | 支持缩放、旋转、平移、仿射变换 | 中 |
| FR-006 | 图像阈值化 | 支持二值化、反二值化、截断、归零、自适应阈值、Otsu 自动阈值 | 中 |
| FR-007 | 平滑/滤波 | 支持高斯模糊、中值滤波、均值模糊、双边滤波 | 中 |

### 2.3 图像增强与形态学模块

| 需求编号 | 功能名称 | 功能描述 | 优先级 |
| :--- | :--- | :--- | :--- |
| FR-008 | 形态学变换 | 支持腐蚀、膨胀、开运算、闭运算 | 中 |
| FR-009 | 直方图均衡化 | 增强图像对比度 | 中 |

### 2.4 交互与显示功能

| 需求编号 | 功能名称 | 功能描述 | 优先级 |
| :--- | :--- | :--- | :--- |
| FR-010 | 文件上传 | 支持 JPG/PNG 格式图片上传 | 高 |
| FR-011 | 双栏对比布局 | 左侧显示原图，右侧显示处理结果图 | 高 |
| FR-012 | 标注可视化 | 在处理结果图上用矩形框/轮廓线标注检测目标 | 高 |
| FR-013 | 下载结果 | 提供下载按钮保存处理后的图片 | 中 |
| FR-014 | 暂存功能 | 支持临时保存处理结果，可将暂存图片作为新原图继续处理 | 中 |
| FR-015 | 多语言支持 | 支持中文/英文界面切换 | 低 |

### 2.5 参数调节功能

| 需求编号 | 功能名称 | 功能描述 | 优先级 |
| :--- | :--- | :--- | :--- |
| FR-016 | Canny阈值调节 | 提供阈值1和阈值2的滑动条 | 高 |
| FR-017 | 人脸检测参数调节 | 提供缩放因子、最小邻域数、最小人脸尺寸、眼睛检测、肤色检测、置信度等参数 | 高 |
| FR-018 | 目标检测置信度调节 | 提供置信度滑动条 | 高 |
| FR-019 | 图像处理参数调节 | 根据选择的功能动态显示对应参数 | 中 |
| FR-020 | 实时预览 | 参数调整时自动更新处理结果 | 高 |

## 4. 技术要求

### 4.1 技术栈

| 分类 | 技术 | 版本要求 |
| :--- | :--- | :--- |
| 语言 | Python | >= 3.8 |
| 框架 | Streamlit | >= 1.0 |
| 图像处理 | OpenCV | >= 4.5 |
| 测试框架 | pytest | >= 6.0 |
| 容器化 | Docker | >= 20.0 |
| 反向代理 | Nginx | >= 1.20 |

### 4.2 模型要求

- 人脸检测：使用 OpenCV 自带的 Haar Cascade 模型（多分类器融合）
- 目标检测：使用 MobileNet-SSD 预训练模型
- 模型加载：支持多源下载、自动重试机制

### 4.3 部署要求

- 监听地址：`0.0.0.0`
- 默认端口：`8501`（Streamlit），`80`（Nginx）
- 支持容器化部署（Docker + Docker Compose）
- 支持健康检查
- 支持日志持久化

---

## 5. 数据需求

- 支持上传的图片格式：JPG、PNG
- 图片大小限制：建议不超过 20MB
- 日志文件：存储于 `/app/logs/` 目录

## 6. 验收标准

### 6.1 功能验收

1. **边缘检测**：上传图片后，调整 Canny 阈值，右侧能正确显示边缘检测结果
2. **人脸检测**：上传包含人脸的图片，能正确识别并标注人脸位置，支持参数调节
3. **目标检测**：上传包含常见物体（人、车等）的图片，能正确检测并标注
4. **色彩空间转换**：支持灰度、HSV、RGB 转换
5. **几何变换**：支持缩放、旋转、平移、仿射变换
6. **阈值化**：支持多种阈值化方式
7. **平滑/滤波**：支持多种滤波方式
8. **形态学变换**：支持腐蚀、膨胀、开运算、闭运算
9. **直方图均衡化**：增强图像对比度
10. **实时预览**：调整参数时，处理结果在 2 秒内自动更新
11. **下载功能**：点击下载按钮，能正确保存处理后的图片
12. **暂存功能**：支持暂存处理结果，可作为新原图继续处理

### 6.2 界面验收

1. 布局清晰，左右双栏对比显示
2. 参数调节控件位置合理，操作方便
3. 标注结果清晰可见
4. 支持中文/英文界面切换

### 6.3 部署验收

1. Docker 镜像构建成功
2. Docker Compose 一键部署成功
3. 健康检查通过
4. 服务异常自动重启
